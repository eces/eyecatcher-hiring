# 아이캐쳐 EYECATCHER 
## 새로운 개발자를 찾습니다!

### 분야

- 웹 프론트/백앤드 개발
- 풀 스택 개발자 (우대)

### 관련 기술

- Node.js
- Express 3
- Grunt (watch, asset compilation, build, deploy 자동화)
- CoffeeScript (모르시면 오셔서 같이 공부해요)
- LessCSS (모르시면 오셔서 같이 공부해요)

### 인프라

- Amazon EC2
- Amazon S3
- Amazon RDS
- 서버 관리 문서화
    - nginx
    - forever
    - papertrail (log management)

### 만드시는 제품

- 전세계 사진 작가와 고객들을 이어주는 플랫폼
- I18N, I10N 완료
- 사진 업로드, 예약 관리, 검색
